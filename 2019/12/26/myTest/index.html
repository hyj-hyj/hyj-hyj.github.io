idea中常用快捷键

2019-12-26
Ctrl
*  Ctrl+R： 在当前文件中替换
*  Ctrl+J： 插入自定义动态代码模板
*  Ctrl+D： 复制行
*  Ctrl+X： 删除行
*  Ctrl＋P： 显示方法的参数信息
*  Ctrl＋E： 显示最近编辑的文件列表
*  Ctrl＋N或者双击Shift或者Shift+Ctrl+N： 根据文件名快速查找文件
*  Ctrl＋/： 单行注释和配置文件注释
*  Ctrl＋U： 前往当前光标所在的方法的父类的方法 / 接口定义
*  Ctrl＋B： 进入光标所在的方法/变量的接口或是定义处，等效于 Ctrl + 左键单击
*  Ctrl＋F1： 在光标所在的错误代码处显示错误信息
*  Ctrl＋F3： 调转到所选中的词的下一个引用位置
*  Ctrl＋F11： 选中文件 / 文件夹，使用助记符设定 / 取消书签
*  Ctrl＋1,2,3...9： 定位到对应数值的书签位置
*  Alt+Enter： 导入包自动修正
Ctrl+G： 在当前文件跳转到指定行处
Ctrl＋F12： 显示当前文件的结构
Ctrl+H： 显示类结构图
Shift+Ctrl＋/： 多行注释
*  Shift+Ctrl＋U： 字符串变大小写
*  Ctrl+Alt+L： 格式化代码
*  Ctrl+Alt+←（左箭头）： 回到上一个光标处
*  Ctrl+Alt+→（右箭头）： 跳到下一个光标处
Ctrl＋Alt＋T： 可以将代码包在一块内，例如try/catch
Ctrl＋Alt＋V： 自动创建变量也就是=号前的变量定义
*  Ctrl＋Alt＋O： 去除多余的（没用到的）jar包
Shift+Ctrl+J： 将多行变成一行
Shift
*  Shift+F2： 跳转到上一个高亮错误 或 警告位置
*  Shift+F3： 在查找模式下，查找匹配上一个
*  Shift+F11： 弹出书签显示层
*  Shift+Tab： 取消缩进
Alt
*  Alt+Enter： 导入包自动修正
*  Alt+Insert： 代码自动生成，如生成对象的 set / get 方法，构造函数，toString() 等
*  Alt+←（左箭头）： 向左切换当前已打开的窗口中的子视图
*  Alt+→（右箭头）： 向右切换当前已打开的窗口中的子视图
